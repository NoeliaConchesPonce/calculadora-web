# Calculadora Web (DAW)

Este es el proyecto de la asignatura Despliegue de Aplicaciones Web, desarrollado con Git.

## Instrucciones de uso

1. Clonar el repositorio.
2. (Opcional) Abrir `index.html` en un navegador para ver la interfaz.
3. Para probar la funcionalidad, es necesario desplegar los archivos en un servidor web con soporte PHP (como XAMPP o Apache).
